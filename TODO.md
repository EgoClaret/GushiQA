# 基于知识图谱的古诗词问答系统 - 实现任务清单

## 博客概述
构建一个基于知识图谱的古诗词问答系统，使用Python + Flask + Neo4j + MySQL技术栈。系统能够回答关于古诗词的各种问题，如作者、朝代、作品等。

## 技术方案
- **技术栈**：Python + Flask + Neo4j + MySQL + HTML/CSS/JavaScript
- **核心功能**：
  - 用户注册登录系统
  - 古诗词知识图谱构建
  - 智能问答系统
  - 意图识别与实体识别
  - 答案生成与展示
